## Overview
Real-time American Sign Language recognition system using TensorFlow, OpenCV, Flask, and MobileNetV2.

## Features
- Real-time webcam prediction
- ASL alphabet recognition
- Prediction smoothing
- Sentence formation
- Word recommendations
- Text-to-Speech
- Progress bar UI
- Flask web interface

## Tech Stack
- Python
- TensorFlow/Keras
- OpenCV
- Flask
- HTML/CSS/JavaScript
